## 文档更新历史

### v1.0.0 (2026-04-24)

#### 新增

- 新增用户登录接口 `POST /auth/login` - 用户通过邮箱和密码登录，获取 access_token
- 新增用户注册接口 `POST /auth/register` - 新用户注册并获取 access_token
- 新增获取当前用户信息接口 `GET /users/profile` - 获取当前登录用户的详细信息
- 新增更新用户信息接口 `PUT /users/profile` - 更新当前登录用户的信息
- 新增修改密码接口 `POST /users/change-password` - 修改当前用户的密码
- 新增上传头像接口 `POST /users/upload-avatar` - 上传用户头像
- 新增获取用户详情接口 `GET /users/:id` - 获取指定用户的公开信息
- 新增获取当前用户的宠物列表接口 `GET /pets` - 获取当前登录用户的所有宠物列表
- 新增获取宠物详情接口 `GET /pets/:id` - 获取指定宠物的详细信息
- 新增添加宠物接口 `POST /pets` - 添加新的宠物
- 新增更新宠物信息接口 `PUT /pets/:id` - 更新宠物信息（只能更新自己的宠物）
- 新增删除宠物接口 `DELETE /pets/:id` - 删除宠物（只能删除自己的宠物）
- 新增获取帖子列表接口 `GET /posts` - 获取帖子列表，支持分页、筛选、排序
- 新增获取帖子详情接口 `GET /posts/:id` - 获取帖子详情，会增加浏览次数
- 新增发布帖子接口 `POST /posts` - 发布新帖子
- 新增更新帖子接口 `PUT /posts/:id` - 更新帖子（只能更新自己的帖子）
- 新增删除帖子接口 `DELETE /posts/:id` - 删除帖子（只能删除自己的帖子）
- 新增获取帖子评论列表接口 `GET /comments/post/:postId` - 获取指定帖子的所有评论
- 新增获取评论详情接口 `GET /comments/:id` - 获取指定评论的详情
- 新增发表评论接口 `POST /comments` - 发表新评论或回复评论
- 新增更新评论接口 `PUT /comments/:id` - 更新评论（只能更新自己的评论）
- 新增删除评论接口 `DELETE /comments/:id` - 删除评论（只能删除自己的评论）
- 新增上传文件接口 `POST /files/upload` - 上传文件（图片、视频等）
- 新增删除文件接口 `DELETE /files/:id` - 删除自己上传的文件
- 新增搜索接口 `GET /search` - 全局搜索，支持搜索帖子、用户、标签
- 新增获取通知列表接口 `GET /notifications` - 获取当前用户的通知列表
- 新增标记通知已读接口 `PUT /notifications/:id/read` - 标记指定通知为已读
- 新增标记所有通知已读接口 `PUT /notifications/read-all` - 标记当前用户的所有通知为已读
