## 🔧 开发规范

### 代码规范
- 使用 TypeScript 严格模式
- 遵循 React Hooks 规范
- 使用 Tailwind CSS 进行样式开发
- 统一的错误处理和加载状态

### 提交规范
- `feat`: 新增功能
- `fix`: 修复 bug
- `refactor`: 代码重构
- `docs`: 文档更新
- `test`: 测试代码

### 分支管理
- `main`: 主分支（生产环境）
- `dev`: 开发分支（集成测试）
- `feature/xx`: 功能开发分支
- `bugfix/xx`: 缺陷修复分支
